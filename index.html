What is a Design System?

What is a Design System?

A design system is a set of interconnected patterns and shared practices coherently organized to serve the purpose of a digital product.
– Alla Kholmatova, Design Systems book

What is a Design System?

It enables...

  • Common design language
  • Consistent cross-product UI and usability
  • Built-in a11y
  • Cross-discipline collaboration
  • Faster time-to-market

What a Design System is NOT?

  • Just a collection of components
  • Silver bullet solution to all UX problems
  • Just a toolkit for Designers

Big companies using Design Systems

  • Airbnb

  • Atlassian
  • Github
  • and many more...

  • Guidelines for contribution
  • Unit testing with jest
  • Agree on commit convention
  • Don't use project aliases
